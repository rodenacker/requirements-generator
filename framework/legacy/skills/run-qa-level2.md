# run-qa-level2.md

**Purpose:** Level 2 Q&A — summarised / suggested-resolution mode. Presents grouped findings with the Unicorn's recommended resolution per group; consultant accepts, edits, or counter-proposes.

**Inputs:** Level 1 captured answers + still-unresolved findings, current spec, Q&A character.

**Outputs:** resolution decisions per finding for `reconcile.md`.

**Used by:**
- `framework/agents/requirements-qa/agent.md` (Level 2 internal phase).
- `framework/agents/design-qa/agent.md` (Level 2 internal phase).

**Used how:** Runs only after Level 1 has compressed the question set. Designed for cases where individual questioning would be too slow.
