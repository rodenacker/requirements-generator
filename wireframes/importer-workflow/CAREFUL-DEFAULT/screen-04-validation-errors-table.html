Validation errors

Primary content

Reachable only when the parent file log is in Failed state. This view shows the row-level reasons validation failed. Approved or Imported file logs do not surface this screen.

Column definitions and row data are returned by the server at runtime. The table below renders the system-defined columns and the per-row error data as the system supplies them — the row shape is intentionally not pre-declared, so the Importer sees whatever the validation engine produced for this particular file setting.
